# Markdown Guide

GraphNotes uses Markdown for note content. Here's a quick reference.

## Text Formatting

- **Bold**: `**text**`
- *Italic*: `*text*`
- ~~Strikethrough~~: `~~text~~`

## Lists

Unordered:
- Item one
- Item two
  - Nested item

Ordered:
1. First
2. Second

## Code

Inline: `code here`

Block:
```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

## Links

- External: `[text](url)`
- Internal: `[[Note Title]]`
