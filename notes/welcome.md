# Welcome to GraphNotes

This is your first note! GraphNotes is a knowledge management app that visualizes your notes as a connected graph.

## Key Features

- **Graph Visualization**: See connections between your notes
- **Bidirectional Links**: Create links between notes with [[wikilinks]]
- **Named Relationships**: Define semantic relationships between notes

Check out the [[Core Concepts]] note to learn more about how to use the app effectively.
